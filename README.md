# Personal Expense Tracker - Java

A simple command-line Java application to track daily expenses. Stores data in a local file (`expenses.txt`).

## Features
- Add new expenses
- View all expenses
- Delete specific expense
- Data persistence using file

## Technologies
- Java
- File I/O
- OOP

## Run Instructions
1. Clone the repo
2. Compile: `javac src/*.java`
3. Run: `java src/Main`
